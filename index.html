Real-Time OpenGL Path Tracer
Scott Kauker



Description

Rendering real-time path traced images in a glsl shader to create interesting global lighting scenes.

Features

  • Monte-Carlo ray selection with cosine-weighted hemisphere distributions
  • Adjustable samples per-pixel and ray bounce length
  • Optionally fully render a scene by pressing F, fully resolving in seconds
  • Supports spheres, planes, and kind of cylinders, each with their own properties, color, and light emittance
  • Optional gaussian blur for noise elimination
  • Allow a free-look camera to move around the active scene.
  • Scene can be adjusted through uniform inputs
  • Reflections
  • Uses OpenGL spec 3.2, so it is widely compatible with many consumer PCs
